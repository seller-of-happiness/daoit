# Функциональность поиска в документах

## Что было реализовано

### 1. Добавлено поле поиска в интерфейс документов
- Поле поиска расположено под заголовком и кнопками действий
- Включает иконку поиска и кнопку очистки
- Поддерживает поиск по Enter или автоматически через 500мс после остановки ввода
- Кнопка "Сбросить" для быстрой очистки поиска

### 2. Реализована логика поиска в store
- Параметр `search` уже был предусмотрен в `IListDocumentsPayload`
- Добавлена передача поискового запроса в API
- Добавлен индикатор загрузки (`isLoading`)
- Store отправляет поисковый запрос на сервер через `POST /api/documents/list/`

### 3. Улучшен UX
- Дебаунс поиска (500мс) для снижения нагрузки на сервер
- Сброс поиска при навигации между папками
- Отображение результатов поиска с количеством найденных элементов
- Адаптивные сообщения о пустом состоянии (разные для поиска и обычного просмотра)

## Как использовать

1. Откройте раздел "Документы"
2. Введите поисковый запрос в поле "Поиск документов..."
3. Поиск выполнится автоматически через 500мс или при нажатии Enter
4. Результаты отобразятся в таблице с индикатором количества найденных элементов
5. Для очистки поиска используйте кнопку "×" или "Сбросить"

## Технические детали

### API
- Эндпоинт: `POST /api/documents/list/`
- Параметр: `search` (string)
- Поиск работает на стороне сервера

### Компоненты
- `DocumentsInterface.vue` - основной интерфейс с полем поиска
- `documentsStore.ts` - логика взаимодействия с API

### Стили
- Адаптивный дизайн
- Поддержка темной темы
- Интеграция с существующей дизайн-системой PrimeVue
