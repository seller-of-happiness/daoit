# Реализация Chat API

## Обзор

В проекте реализованы все необходимые API endpoints для работы с чатами, группами и каналами в соответствии с предоставленной спецификацией.

## Реализованные API Endpoints

### 1. Создание диалога
- **Endpoint**: `POST /api/chat/chat/dialog/`
- **Описание**: Создает или возвращает существующий диалог с пользователем
- **Реализация**: `chatStore.createDialog(userId)`
- **Статус**: ✅ Реализовано

### 2. Создание группы  
- **Endpoint**: `POST /api/chat/chat/group/`
- **Описание**: Создает новую группу
- **Реализация**: `chatStore.createGroup(payload)`
- **Статус**: ✅ Реализовано

### 3. Создание канала
- **Endpoint**: `POST /api/chat/chat/channel/`
- **Описание**: Создает новый канал
- **Реализация**: `chatStore.createChannel(payload)`
- **Статус**: ✅ Реализовано

### 4. Добавление участников
- **Endpoint**: `POST /api/chat/chat/{id}/add-member/`
- **Описание**: Добавляет участников в группу/канал (отправляет приглашения)
- **Реализация**: `chatStore.addMembersToChat(chatId, userIds)`
- **Статус**: ✅ Реализовано

### 5. Удаление участников
- **Endpoint**: `DELETE /api/chat/chat/{id}/remove-member/?user_id={uuid}`
- **Описание**: Удаляет участника из чата
- **Реализация**: `chatStore.removeMemberFromChat(chatId, userId)`
- **Статус**: ✅ Реализовано

### 6. Обновление информации о чате
- **Endpoint**: `PATCH /api/chat/chat/{id}/`
- **Описание**: Обновляет информацию о группе/канале
- **Реализация**: `chatStore.updateChat(chatId, payload)`
- **Статус**: ✅ Реализовано (исправлен метод с PUT на PATCH)

## Структура данных

### Чат (IChat)
```typescript
interface IChat {
    id: number
    type: 'dialog' | 'group' | 'channel'
    title: string
    description: string
    icon: string | null
    members: IChatMember[]
    invites: IChatInvite[]
    created_by: IUser
    // ... дополнительные поля
}
```

### Участник чата (IChatMember)
```typescript
interface IChatMember {
    user: IUser
    is_admin: boolean
    joined_at: string
}
```

### Приглашение (IChatInvite)
```typescript
interface IChatInvite {
    id: number
    invited_user: IUser
    is_accepted: boolean
}
```
