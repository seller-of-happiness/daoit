# Отладка сортировки чатов

## Проблема
Сортировка чатов по времени последнего сообщения не работает корректно. Чаты не поднимаются наверх при получении новых сообщений.

## Добавленное логирование

### 1. Функция сравнения чатов (`compareChatsByLastMessage`)
- Показывает время последнего сообщения или создания каждого чата
- Отображает результат сравнения между чатами
- Включает безопасную обработку дат с проверкой валидности

### 2. Метод сортировки (`sortChatsByLastMessage`)
- Показывает количество чатов для сортировки
- Выводит состояние чатов до и после сортировки
- Отображает ID, название, время последнего сообщения и время создания

### 3. Обработка новых сообщений (`handleNewMessage`)
- Логирует детали каждого нового сообщения
- Показывает, обновляется ли чат с новым сообщением
- Отслеживает процесс сортировки после получения сообщения

### 4. Загрузка чатов из API
- Показывает структуру данных, получаемых с сервера
- Отображает информацию о последних сообщениях в каждом чате

## Как использовать логирование

1. Откройте DevTools в браузере (F12)
2. Перейдите на вкладку Console
3. Отфильтруйте логи по тексту `[CHAT SORT]`
4. Выполните действия:
   - Отправьте сообщение в чат
   - Получите сообщение от другого пользователя
   - Обновите страницу с чатами

## Возможные проблемы для проверки

1. **Неправильный формат даты**: Проверьте, что `created_at` в правильном формате
2. **Отсутствует last_message**: Убедитесь, что API возвращает объект `last_message`
3. **Неправильное обновление чата**: Проверьте, что `last_message` обновляется при получении нового сообщения
4. **Проблемы с реактивностью**: Убедитесь, что Vue правильно отслеживает изменения массива чатов

## Удаление логирования

После решения проблемы удалите все строки с `console.log`, содержащие `[CHAT SORT]`.
