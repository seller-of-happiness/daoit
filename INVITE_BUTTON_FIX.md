# Исправление кнопки "Пригласить пользователей"

## Проблема
Пропала кнопка приглашения пользователей в группах и каналах.

## Причина
Проблема была в неправильном сравнении ID пользователей при проверке прав администратора. ID могли иметь разные типы или форматы (string vs number).

## Решение
Заменили ручное сравнение ID на использование готового адаптера `ChatAdapter.isUserAdmin()`, который корректно обрабатывает различные форматы ID.

### Измененные файлы:

#### ChatHeader.vue
- Заменена логика проверки `canInviteUsers`
- Добавлен импорт `ChatAdapter`
- Удален отладочный код

#### ChatMembersManagement.vue  
- Заменена логика проверки `canManageMembers`
- Добавлен импорт `ChatAdapter`

## Теперь кнопка появляется когда:
1. Чат является группой или каналом
2. Текущий пользователь является администратором чата

## Альтернативные способы приглашения:
- Через кнопку "Управление чатом" → "Пригласить участников"
- При создании нового чата с опцией "Пригласить участников после создания"

## Тестирование
1. Создайте группу или канал
2. Убедитесь, что создатель автоматически становится администратором
3. Проверьте появление кнопки "Пригласить пользователей" в заголовке чата
4. Проверьте работу приглашения через диалог управления чатом
